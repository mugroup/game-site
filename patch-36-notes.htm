Redshift: Version 3.6 Patch Notes

Baraphor
March 29, 2016
Patch Notes, Redshift

Patch 3.6E Notes


Changes


*IMPORTANT* We have swapped the positions of buttons in many windows, please look before you click.

 

  • Changed the window for recruiting captains, it will be faster now.
  • AutoBurn window no longer pops up a window when you have selected 60 items, instead it is built into the existing window.
  • You can now claim expired gifts using credits.
  • Titanium and Selenium have been added as possible Operation rewards.
  • Rifts will now show a message when you are fighting a Boss Wave.
  • We removed the 50 Alliance point cost to activate in your alliance, you can also reactivate at anytime now.
  • When you extend a Salvage module, the game will ask if you would like to share on the alliance discussion board.
  • We have gone over the early sectors and adjusted the difficulty curve.
  • We have adjusted Rift T3 and T4.
  • You can now select if you want to burn an xp card to a ship or crew member.
  • At the end of a Rift you will see a different window now based on winning or losing.
  • We added a new repeatable mission for Rift wheel spins
  • We added a new daily mission for a ship's active ability.
  • We added a visual indicator when a ship is being repaired to the fleet hub.
  • When in the rift, the selection for ships will not try and pick the best ship for you to deploy, rather than the next ship.
  • We added names to the Capital Ship side window for each tab.
  • When you are invited to a Rift, you can now select your fleet.
  • We made come changes to the Events and how they are broken up.
  • We added a swap button for ships, escorts and interceptors next to the remove button.
  • We added a loading screen when the game boots up

Bug Fixes

 

  • The Liberator ability was not granting kills when a enemy died.
  • The sale window will now check the discounted value for credits when checking to see if you have enough to purchase a sale.
  • When purchasing Carbon, it will now purchase the difference and not the total.
  • When you finish a Rift, it will now no longer ask you to burn items.
  • Fixed some display errors with mission rewards, the Server was providing you with the correct amount.
  • You can no longer burn a ship to itself if it was in your inventory.
  • You can now leave sector 1,2 and 3 after you finish the tutorial
  • You can now view all fleets when looking at another players fleets, all of the time.
  • Capital Ship weapons no longer ignore being stunned.
  • Fixed a few left over spelling mistakes.
  • Fixed a crash when you accept a Rift invite and you had the side window open.
  • When you select an individual fleet to view from the leaderboard, you will now be taken to that fleet.
  • When a ship is destroyed in the Rift, it health bar will no longer fill up again at the end of a wave.
  • Fixed a bug were when you joined a rift late you could end up one wave ahead.
  • We fixed a bug which would allow you to get a ship beyond level 100.
  • We fixed a bug were the base placement locations would not disappear.
  • The event that requires you to upgrade a command center will now respect the upgrade timer.
  • Fixed a bug on android were your alliance lab would say idle even when you were decoding a schematic.

Patch 3.6B Notes


Bug Fixes

 

  • You can now select a crew autoburn target again.
  • Event where you need to complete a planetary raid now requires 3 stars.
  • The VIP window now works correctly
  • In the defense mode if you get 3 stars, you will also see 100% completion, always.
  • Fixed the bug where you saw x1000 Ninja ships instead of Captain Briggs.
  • The Capital Ship autoburn should not longer select any schematics for burning.
  • Fixed bug where ships would attack themselves or friends.
  • The wheel will open to which one has spins.
  • In the Salvage window it will only highlight the button if you have harvests.
  • In the Salvage Activation window it will only highlight if you have one to activate.

Balancing

  • Equipment now percentage based increases, instead of fixed value.
  • The daily mission for 20 platinum wheel spins reward has been adjusted, it can no longer crit up to a higher rarity.
  • The common Capital SHip active has a target cap now of 20 now.
  • Added an additional event step for the base.
  • Balanced a few ship stats.

UI Tweaks

  • The CS is now the default selected ship for deployment.
  • The Crew Abilities panel will sort based on abilities being used now, unused will always be shown first.
  • You can now click on a currency in the Main Ui to see what it does.

Patch 3.6A Notes


Features
Key Features

  • New and Improved tutorial
  • Salvage System
  • Event System
  • Additional Capital ships
  • Rifts Tier 4
  • Sectors Overall
  • New Missions
  • Overhauled Abilities, Ships and Interceptors
  • Capital Ship Passive Equipment now uses Titanium for upgrading.
  • Graphical Improvements
  • Performance Improvements
  • Immortal Captains can now support up to 8 crew members per fleet.
  • Players can use all active abilities for Crew now.
  • VIP Rewards

Tutorial

  • New and Improved tutorial
  • Covered more in less
  • Focused on teaching, rather than showing.
  • R.E.V.A will play the role of your advisor throughout the tutorial and events.

Salvage System

  • We have added a new Alliance mechanic for collecting resources.
  • This mechanic can be used to collecting items to help the game and build additional capital ships
  • The Salvage will generate Unobtainium for players to collect.
  • Players can collect or purchase Salvage Modules.
  • These modules can be decoded and re-roll to get good resources for harvest.
  • Players add there module to the Salvage window for their alliance allowing everyone to harvest from them.

Event System

  • We have created a brand new event system for the game, currently it covers the early game, however we are going to expand this in future patches.
  • Events will contain multiple goals, which in turn have multiple objectives to complete.
  • Events also have a story and characters and move along with you.
  • Additional Capital ships
  • Players can now have up to 10 Capital Ship in their armada.
  • Currently it is limited to one per fleet
  • Added a new building to the Alliance, called the Shipyard where players can construction Capital Ships using Unobtainium collected from Salvaging.

Rifts

  • Rift Tier 4 is now open
  • 40 waves
  • Significantly more difficult that Tier 3
  • More Loot

Sectors

  • Sectors have been redesigned, they are more balanced and have an improved difficulty curve.
  • Sector Rewards have been increased.
  • There is a new Sector type, Defensive Battle, where a player defends pylons similar to the Rift.
  • Sectors provide a large reward the first time they are completed with three stars, existing players will be able to claim these rewards by beating a sector with 3 stars.
  • A new story that follows you through the Federation attempts to Destroy the Galactic Union.

VIP System

  • 10 tiers, daily rewards and passive bonuses.
  • Including
  • Bonus resources collected.
  • Free repairs and building time skips.
  • Boost All Button for operations.
  • Plating and speed bonuses for a player's ships.
  • Increased experience from experience cards earned.
  • Discounts on sales.
  • Many different daily prizes including resources and tokens.
  • Free re-rolls for crew members, schematics and salvage modules.
  • The cold matter on the platinum wheel has been replaced by a 60m VIP activator.
  • The cobalt on the rift wheel has been replaced by a 15 VIP points.
  • And More...

Missions

  • New Upgrade Missions
  • Replaced the existing missions for upgrading ships to be more reflective that players should spread upgrades out across all their ships.
  • Missions have been broken down into into have x ships of level y, where x is 1, 3, 5, 7, 9, 11 or 13 and y is level 5-100, incrementing by 5 levels each time.
  • This results in more missions overall, however the rewards will be comparable to what was earned previously.
  • Existing players will be able to claim these new missions.

  • The Capital Ship will no longer appear as a possible target to burn items too while in a battle.

  • Improved the background graphics in the game.
